# Java Linked List Processor

A Java program that constructs and processes a linked list from an input file.

## Features
- Reads integer data from a file
- Builds a custom linked list
- Uses a custom Node class
- Processes linked list values based on conditional logic
- Writes the processed list to an output file

## Concepts Used
- Linked Lists
- Object-Oriented Programming
- File Input/Output
- Classes and Objects
- Data Structures

## Technologies
- Java
- Eclipse / IntelliJ / VS Code

## Files
- `LinkedList.java` - linked list implementation
- `Node.java` - node class
- `LinkedListTester.java` - main tester file

## What I Learned
This project helped me understand linked list construction, node references, file handling, and object-oriented programming in Java.
